# Python Data Analysis

Python was used to perform exploratory data analysis (EDA) on the cleaned E-Commerce dataset.

## Libraries Used

- Pandas
- Matplotlib

## Analysis Performed

- Loaded and inspected the cleaned dataset
- Checked dataset structure and data types
- Analyzed missing and duplicate values
- Performed exploratory data analysis
- Analyzed sales and order trends
- Analyzed customer and product performance
- Created visualizations to understand business patterns
- Exported the processed Final E-Commerce Dataset

## Input

The Clean Dataset created during the Excel data cleaning stage was used for Python analysis.

## Output

A processed Final E-Commerce Dataset was exported after Python analysis.

The dataset files are available through the links provided in the `01_Dataset` folder.
