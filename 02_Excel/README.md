# Excel Data Cleaning

Excel was used for the initial data cleaning and preparation of the E-Commerce dataset.

## Key Tasks Performed

- Checked and handled missing values
- Checked duplicate records
- Corrected data types and formatting
- Cleaned date and time columns
- Created additional columns for analysis
- Prepared the Clean Dataset for further analysis

## Output

The cleaned dataset was used as the primary dataset for:
- Python Analysis
- PostgreSQL / SQL Analysis
- Power BI Dashboard Development

The complete Clean Dataset is available in the `01_Dataset` section.
